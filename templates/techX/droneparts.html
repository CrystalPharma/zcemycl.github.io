Setup

  1. In Ubuntu 16.04, go to command line and type,
    2. 
sudo apt install ros-kinetic-tello-driver
  2. Turn on Tello drone.
  3. Connect to Tello drone's wifi.
  4. Run rosnode,
    5. 
roslaunch tello_driver tello_node.launch

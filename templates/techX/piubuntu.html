Pi Setup by Ubuntu

Flash Pi by Ubuntu

  1. Install Raspberry Pi Imager. 
    2. 
sudo snap install rpi-imager
  2. Insert your sd card in card reader.
  3. Open Raspberry Pi Imager. Select Raspberry Pi OS Full as OS. Select Generic-SD_MMD_MS_PRO as Storage. Press Write.

Remote Desktop Protocol

  1. Enable SSH via, 
    2. 
sudo raspi-config
Interface Options → SSH
  2. Install xrdp in Raspberry Pi.
    3. 
sudo apt-get install xrdp
sudo service xrdp start (stop|restart|status)
  3. In Ubuntu, install remmina
    4. 
sudo apt install remmina
  4. Settings in Remmina
    5. 
Server: 192.168.1.xxx
User name: pi
User password: raspberry (default)
Resolution: Use client resolution
Color depth: GFX RFX (32 bpp)

React Basics

Setup

After running create-react-app,

  1. All packages are listed in package.json.
  2. Delete all src files except index.js.
  3. Structure of the project,
    4. graph TD Y[Project] --> Z[index.js]; Y --> A; Y --> X[public]; X --> UU[index.html]; Y --> W[package.json]; A[src] --> B[components]; B --> C[App.js]; B --> D[CustomButton]; D --> E[CustomButton.css]; D --> F[CustomButton.js]; D ==> C; C ==> Z; Z ==> UU;

Syntax

  1. index.js (Don't need to touch most of the time)
    2. 
import React from "react"
import ReactDOM from "react-dom"
import App from './components/App'

const reactContentRoot = document.getElementById("root")

ReactDOM.render(<App/>, reactContentRoot)

    3. 
import React from "react"
import customButton from "./customButton"

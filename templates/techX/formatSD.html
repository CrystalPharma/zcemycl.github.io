Format SD Cards

Steps:
  1. Insert SD card into the reader.
  2. In windows, press Win + R. Type diskpart to bring the terminal.
  3. Type list volume.
  4. Select the right volume (usually the last one). select volume n
  5. Format it. For SD which will contain px4, format fs=fat32. For SD being flashed by raspi, format fs=ntfs.
  6. After 100 percent completed, successfully formatted, type exit.

Elastic IP address

Background

Again, this is a rebranding from aws. Elastic IP (original name is Static IP) is an IPv4 address you need to buy from ISP or AWS or other internet company, so that your domain name can be bound to stably over time.

Steps

  1. Navigate to EC2 dashboard interface.
  2. Scroll down the left sidebar, and find Network & Security.
  3. Press Elastic IPs → Allocate Elastic IP address.
  4. Select Amazon's pool of IPv4 address → Allocate.
  5. (Optional) Actions: Associate to EC2 Instance.

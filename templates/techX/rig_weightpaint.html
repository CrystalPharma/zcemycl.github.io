Rigging and Weight painting

One piece of Mesh

Weight distribution: red(more) to blue (less)

  1. Object Mode → Select body then bone → Ctrl+P → Automatic Weights
  2. Select bone → Alt+P → Disconnect Bone (can remain parent) / Clear Parent
  3. Need poles for leg and hip joint, and leg and foot joint. Remember to uncheck deform.
  4. leg needs Under Pose Mode, add Inverse Kinematics → Target and Pole Target are Amature, Bone 1 is foot pole and Bone 2 leg pole. Pole Angle needs adjustment if any distortion, chain length = 2.
  5. foot needs add Copy Rotation, Invert Y and Z, uncheck Inherit Rotation. Target: Armature, and Bone footpole, Target and Owner are Local Space.

Multi pieces of Mesh

Useful shortcut keys
  1. Tab: switch mode between Object and Edit
  2. Alt+Z: X-ray mode
  3. G: Grap (Move), can expand a point in or out.
  4. R: Rotate
  5. S: Scale
  6. A: Select All
  7. Shift+A: Add object
  8. Shift+D: Duplicate object
  9. Shift+C: Reset Cursor
  10. Ctrl+A: Apply
  11. Shift+Right Click: relocate cursor
  12. X,Y,Z: Can lock to specific axis
  13. Numberpad 3:X,1:Y,7:Z align to axis
  14. Shift+A → Amature: add bone
  15. E: Extrude (bone)
  16. O+?: Cancel ?
  17. Ctrl+Tab: Next next mode
  18. Edit → Preferences → Add-ons ⋺ Search: enable add-ons
  19. Select Amature Layer → Ctrl+P → With Automatic Weights: Assign automatic weight
  20. Under Object Mode → Select Skin layers you need and Armature → Under Pose Mode → Shift Select that one specific bone → Ctrl+P → Bone: Option2 Assign Bone if automatic weight does not work
  21. Ctrl+P → Keep offset: pairing and connecting but keep distance.
  22. Ctrl+A → Rotations → Shift+D → S → X → -1: Copy the object and reflect along X?
  23. Shift D → G+X → Remain Select → Right click → Mirror → Along X-axis: duplicate and mirror along X

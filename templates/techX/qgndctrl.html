QGndCtrl and PX4 Configuration

Flashing the flight controller (FCU)

Steps:
  1. Insert the SD card with FAT32 filesystem into your flight controller (FCU).
  2. Download and install QGroundControl in your laptop.
  3. Open the QGroundControl software. Open the Vehicle Setup tab and select the Firmware menu.
  4. Download the modified version of PX4 by CopterExpress for the Clover drone. Download the latest stable version v1.8.2-clever.10
  5. Connect your flight controller to your PC over USB.
  6. Wait for the FCU to connect to QGroundControl
  7. Select PX4 Flight Stack (Firmware Setup) in the right bar.
  8. Check Advanced Settings checkbox.
  9. Select Custom fireware file from dropdown menu.
  10. Press OK and select the file v1.8.2-clever.10.

Configuring the flight controller

Steps:
  1. After flashing, the parameters will be shown up, such as Airframe, Radio, Sensors, Flight Modes, etc.
  2. Open Vehicle Setup tab. Select the Airframe menu.
  3. Select the Quadrotor X airframe type.
  4. Select the Generic Quadrotor X from the dropdown menu.
  5. Return to the top of the list and press Apply and Restart button, confirm by pressing Apply.
  6. Wait for the settings to be applied and for the flight controller to restart.
  7. Go to Parameters. Setting the configuraions as stated in link 2.

TroubleShooting

  • Rebooting causes resetting, the airframe is reset. This can be due to the SD card undetected. Your SD card can be malfunctioning. Change another.
